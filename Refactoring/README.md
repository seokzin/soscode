## 🔎 Keywords

### VScode shortcuts

- 동일 텍스트 선택

  - `ctrl + D` : 동일한 텍스트 추가 선택. 부분 수정에 유용
  - `ctrl + K` : `ctrl + D`로 선택된 텍스트 취소
  - `ctrl + F2` : 동일한 텍스트 전부 선택

- 리팩토링
  - `ctrl + shift + R` : 리팩토링 옵션
