# Nest.js

## Goals

- Angular, Spring MVC 패턴의 간접 이해
- 백엔드와 수월한 협업
- OOP, IoC, DI 등의 개념 실전 이해
- Test (Unit, E2E)

## Keywords

- module
  - 데이터 구조 구축의 시작점. 메타 데이터 제공
- controller
  - URL을 받와 함수 리턴
- service
  - 실제 비즈니스 로직 구현
- DTO (Data Transfer Object)
  - 데이터 전송을 위한 객체
  - 효율적 Validation
- DI (Dependency Injection)
  - 필요한 객체를 직접 생성이 아닌 외부로부터 주입받는 기법
  - DIP 법칙과 함께 의존성을 줄여주는 기법
